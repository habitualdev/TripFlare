# TripFlare

**Slackbot for monitoring external malware databases for uploads of novel malware.**

Proper functionality requires at least:
 - Slack Bot and App token
 - At least one API token

<hr>
Example Yaml config

```
slack:
        bot_token: 
        app_token: 
api:
        virus_total: 
        vx:
			username:
			password:
        malware_bazaar: 
        hybrid_analysis:        
```
